轮胎售后服务线上平台

面向戴姆勒中国客户，提供轮胎售后线上服务，系统分为微信平台和经销商管理网站，客户在微信平台下单订购轮胎，经销商在管理网站上备货，管理订单，监控物流，查看客户信息，从供应商处增加库存，给客户发送优惠卷等，系统覆盖戴姆勒在中国所有经销商。 该系统的Axure产品原型以及业务流程图可以从 GitHub 上下载，见其中的“轮胎售后服务线上平台”目录。

微信平台产品原型截图

1.  选择车型车系

2.  根据车型车系展示匹配的轮胎列表

3.  从轮胎列表，进入某一轮胎详情页面

4.  购物车页面，在该页面可以选择不同的下单方式

5.  订单列表页面

经销商管理网站原型截图

1.  轮胎主数据管理页面，这些数据可以从德国戴姆勒的服务器同步得到

2.  经销商收到轮胎服务订单后，可以补充自己的库存，将轮胎加入购物车，这种操作是将仓库中的轮胎运送到经销商处；如果仓库中某种轮胎缺货或者数量不足，经销商可以申请从供应商处采购，即“订货”操作。

3.  经销商可以查看购物车中的轮胎，并选择其中几个或全部下单，下单后仓库会把相关的轮胎运送到经销商处。

4.  经销商可以查看订购轮胎相关的订单列表，以及其中某个订单的详情。

5.  经销商可以查看和自己相关的轮胎服务订单（由客户在微信平台下单生成），以及其中某个订单的详情。

业务流程设计

包括用户订购轮胎，付款，经销商从供货商处订购轮胎等流程
